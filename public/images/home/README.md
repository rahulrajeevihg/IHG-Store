# Home page images

Drop AI-generated artwork here. Expected filenames (the home page auto-detects them and
falls back to icons/gradients when missing):

- `hero-1.png`, `hero-2.png`, `hero-3.png` — hero banner artwork (900×700, transparent)
- `ai-assistant.png` — AI assistant mascot (480×480, transparent)
- `product-placeholder.png` — fallback product image (800×600)
- `empty-state.png` — empty-state illustration (400×400, transparent)
- `cat-lighting.png`, `cat-electrical.png`, `cat-plumbing.png`, `cat-hardware.png`, `cat-tools.png` — optional category tiles (240×240, transparent)

See `HOME_IMAGE_PROMPTS.md` in the repo root for the generation prompts.
